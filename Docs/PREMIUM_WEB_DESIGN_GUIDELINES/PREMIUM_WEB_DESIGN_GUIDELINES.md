# PREMIUM_WEB_DESIGN_GUIDELINES

## Purpose
This document defines the mandatory UI/UX standards for every page of the Barber Shop Don d' Alex website. The objective is to create a premium digital experience comparable to luxury brands rather than a generic template.

## 1. Design Philosophy
- Premium before flashy.
- Simplicity before decoration.
- Readability before effects.
- Editorial quality on every page.

## 2. Visual Principles
- Luxury dark mode.
- Large whitespace.
- Strong visual hierarchy.
- No generic layouts.
- Consistent design system.

## 3. Color & Atmosphere
- Matte black backgrounds.
- Soft radial gradients.
- Gold accents only for emphasis.
- Elegant contrast.

## 4. Typography
- Large bold headlines.
- Clear hierarchy.
- Comfortable reading width.
- Consistent spacing.

## 5. Photography
- Real photography whenever possible.
- Premium editorial style.
- Close-up details.
- Professional lighting.

## 6. Layout
- Avoid repetitive sections.
- Use editorial compositions.
- Alternate text and imagery.
- Every section should feel unique.

## 7. Components
Buttons, cards, forms and navigation must be reusable, accessible and visually consistent.

## 8. Motion
Use subtle animations:
- Fade
- Scroll reveal
- Hover transitions
- Smooth scrolling

Never distract from content.

## 9. Accessibility
All pages must satisfy WCAG 2.2 AA.

## 10. Premium Checklist
Before approving a page verify:
- Premium visual quality
- Mobile-first
- Excellent spacing
- Strong hierarchy
- High-quality imagery
- Smooth interactions
- Accessibility
- Fast loading
- SEO preserved

## Notes for Antigravity
Do not generate generic Tailwind landing pages.
Design every page as if it were created by a premium digital agency. Choose the highest perceived quality while preserving performance, accessibility and SEO.
